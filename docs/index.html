‚ùì Frequently Asked Questions

How do I install mods?

Simply drop your mod files into the "mods" folder in your PawCraft instance directory. The launcher will automatically detect and load them.

Where are my game saves stored?

Saves are located in the "saves" folder within your PawCraft instance directory. You can also access them via the launcher's file manager.

Can I use PawCraft with existing Minecraft installations?

Yes! PawCraft can detect and import your existing Minecraft installations automatically.

How do I update PawCraft?

Download the latest version from the official website and replace your existing PawCraft files.
